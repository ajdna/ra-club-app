1
Drop the token file into globals.css

Open handoff/globals.css from this project. Copy the entire :root { … }, .dark { … } and @theme inline { … } blocks. Paste them into your app's src/app/globals.css, replacing the existing color/token section.

IN CLAUDE COWORK — tell it:
"Replace the token/color section in globals.css with the contents of the @theme block from the handoff. Variable names are --emerald, --terra, --cream, --card, --ink, --ink-2, --ink-3, --line, --gold, --good, --warn, --bad."
2
Install the two fonts

The design uses Fraunces (display/headings) + Manrope (UI body). Add them to your Next.js layout so they're available app-wide.

IN CLAUDE COWORK — tell it:
"In layout.tsx, import Fraunces and Manrope from next/font/google. Apply them as CSS variables --font-display and --font-sans on the html tag."

4
Enable dark mode

The .dark override block in globals.css handles all color flips automatically. Just tell Cowork to wire up the toggle.

IN CLAUDE COWORK — tell it:
"Add a dark/light mode toggle that adds/removes the .dark class on the html element. All colours are already defined in globals.css under .dark { … } — no other changes needed."
